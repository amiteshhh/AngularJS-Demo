Welcome to the AngularJS demo.

Demo covers below topics
  • DOM manipulation using pure Javascript
  • DOM manipulation/Data binding using AngularJS
  • Event binding
  • Understanding $scope which acts as a middle man between DOM and data(i.e view and model)
  • Angular Controller to extend $scope with data and event methods required for the view
  • Angular Services to define data and business logic
  • Representing model/data differently in view using Angular Filter
  • Single page application using ui-route
No framework whether AngularJS or jQuery can add new capability to the native JavaScript/HTML feature. They simply put abstraction layer over the core feature to handle the things in somehow more convinient way. For example
  • $http or $.ajax has been created on the top of native xmlHTTP object to handle ajax call.
  • `on` method encapsulated over native `addEventListener` for event binding
